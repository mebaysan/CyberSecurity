[[[A]]] Records -> burası altında hangi siteye istek gelirse hangi ip'ye yönlendirileceği belirtilir. bizde burada kendi ip adresimizi yazabiliriz. bu sayede kendi bilgisyaraımızı server olarak kullandığımız gibi kullanıcılara oyun oynayabiliriz.

mitmf --arp --spoof --gateway <gateway_ip> --target <target_ip> -i <interface> --dns

Kullanıcıyı kendi istediğimiz bir yere yönlendireceğiz

files-> other locations -> var -> www => burada kendimizi server olarak tanıtınca gösterilecek olan html dosyaları var

service apache2 start -> apache servisimizi başlattık bundan sonra kendi bilgisayarımızı bir web sitesi gibi kullanabiliriz.


Özetle:
mitmf konfigürasyonlarından hangi siteye girilirse nereye yönlendirileceğini belirliyoruz.
kendi bilgisayarımızı bir server olarak kullanıyoruz.
mitmf saldırısını yapıyoruz.
