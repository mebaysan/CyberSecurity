    Monitör Mod Nedir?
Etraftaki diğer paketleri de yakalamak için monitör mod kullanılır. Etraftaki diğer paketleri dinlemek için.


    Managed Mod Nedir?
Sadece kendilerinin hedeflendiği bilgileri alır.


    Monitör-Managed Mod Değiştirme
   
1-) AİRMON İLE
airmon-ng start wlan0(veya eth0) # bizi monitör moda geçirir

airmon-ng stop wlan0mon(veya eth0mon) # managed moda geri geçeriz

2-) İFCONFİG İLE
ifconfig wlan0 down # servisi kapatır
iwconfig wlan0 mode monitor # monitör moda geçer
ifconfig wlan0 up # servisi tekrar başlatır
iwconfig wlan0 # bu şekilde kontrol edebiliriz
ifconfig wlan0 down
iwconfig wlan0 mode managed # managed moda geçirdik
ifconfig wlan0 up # servisi tekrar başlattık
